# Serenalunia Discord Music Bot

A Python-based Discord bot that connects to voice channels and plays audio through user commands.

## Features

* Joins Discord voice channels
* Audio playback support
* Command-based interaction
* Automatic background operation
* Custom audio integration

## Technologies

* Python
* Discord.py
* FFmpeg
* Event-driven programming

## Purpose

This project was developed to explore Discord API integration, voice channel functionality, command handling, and audio streaming within a real-world application.

## Installation

1. Install dependencies:

pip install -r requirements.txt

2. Configure your Discord bot token.

3. Run:

python main.py
